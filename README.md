1. 算法与模型
* 算法：本质上是一些数学公式，逻辑规则等，是一种计算过程，可以将数据输入后得到一组输出。
* 模型：是算法的结果，根据学习算法中输入的数据及其计算的结果，可对输入的数据进行预测。
